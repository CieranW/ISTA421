Questions

Be sure to answer all of them.

Question 1

Assign the value 26 to a variable x and create a vector y with the values [2, 36, 1, 98, 124]. Multiply those vectors component-wise and save the result in an object z. Calculate the sum of all elements in z and save your results in a new object named Q1.

Question 2

Generate a sequence from 21 to 400 and a second sequence from 2 to -35. Save your results in new objects named Q2.1 and Q2.2, respectively.

Question 3

Generate a sequence from −3 to 9 by 0.1 steps. Save your results in a new object named Q3.

Question 4

Randomly sample 100 times a total of 20 elements (with replacement) from Q3. Save your results in a new object of class list named Q4.

Question 5

Define two vectors with the following information. First, t includes the strings mon, tue, wed, thu, fri, sat. Second, vector m includes [90, 80, 50, 20, 5, 20]. Bind both vectors column-wise into a Pandas dataframe with 6 rows and 2 columns. Save this a new object named Q5.

Question 6

Create three vectors of length 100 named weight (mean=81, sd=0.26; units=Kg), height (mean=1.76, sd=0.22; units=m), and age (mean=25, sd =5; units=years) by randomly sampling three normal distributions using the stats::rnorm function. Generate a fourth vector gender with the same length as height but by randomly sampling (with replacement) from the following vector ["A", "B", "C"].

Question 7

Create a data.frame by combining the four vectors created above in Question 6 as columns. Add a new column (BMI) with only NAs. Save your results in a new object named Q7.

Question 8

Generate a variable selection (e.g. if/else statement) that contains the result to the logical query of age under 20 or above 30. This new vector should be added as a new column to Q7 named AgeGroup. The new dataset should be saved as Q8.

Question 9

Calculate the “Body Mass Index” (BMI; see the equation below) for all people in the Q8 data.frame. Store the results in the BMI column in the same data.frame. Save the resulting object as Q9.

\[ BMI = \frac{Weight_{(Kg)}} {Height_{(m)}^2} \]

Question 10

Create a function that calculates the BMI (see equation above). It should take only two arguments (height and weight) and it should return a numerical vector. This function must be saved as function_Q10. You must show that this function works (e.g. run function_Q10 using height and weight columns from Q9).

Question 11

Create a function that calculates the mean BMI for a particular gender. This function (function_Q11) must take only two arguments. First, a data.frame (with or without BMI estimates). Second, a target gender (see levels in Question 6). This function must be saved as function_Q11. You must show that the function calculates the mean BMI for a particular gender (e.g. run function_Q11 with gender A).

Question 12

Append a new column (BMIc, see below) to the Q9 data.frame with a new vector that corrects BMI values according to the AgeGroup column. Specifically, the new column (BMIc) should follow the structure: if a participant’s age is within the target age group (TRUE for the AgeGroup column), add 10% to the original BMI. If not, use the original BMI estimate. Save the resulting data.frame as Q12.
